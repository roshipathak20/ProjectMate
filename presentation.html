ProjectMate - Presentation

Project Overview

ProjectMate is a full-stack web application that allows users to add, view, update, and delete tasks. It has:

  • Backend: Node.js, Express, PostgreSQL (hosted on Firebase)
  • Frontend: React (Modern UI)
  • Database: PostgreSQL (Cloud)
  • Testing: Thunder Client

Environment & Tools Used

  • Visual Studio Code for development
  • Node.js and Express for backend
  • React for frontend
  • PostgreSQL database (deployed on Firebase)
  • Thunder Client for API testing

Backend Development
